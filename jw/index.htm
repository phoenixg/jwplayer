播放列表的演示

背景色为灰色：

s1.addParam('bgcolor','#AAAAAA');
播放器会在此元素里面。

背景色为红色：

s1.addParam('bgcolor','#FF0000');
播放器会在此元素里面。

播放器宽高值设为：1000×120；视频大小为：480×360

var s1 = new SWFObject('player.swf','player','1000','120','9');
播放器会在此元素里面。

播放器宽高值设为：200×400；视频大小为：480×360

var s1 = new SWFObject('player.swf','player','200','400','9');
播放器会在此元素里面。

使用XML播放列表（XSPF feed格式）

s1.addParam('flashvars','file=playlist1.xml');
播放器会在此元素里面。

JW Box弹出浮动层


	

		
		

			

				

			

		
 
	

		
	
	
	

	
	
stretching=none，视频分辨率是多少尺寸，播放时就是多少尺寸，不进行拉伸

	
sample1：当播放器窗口小于视频分辨率时，视频会只显示一部分

	
var s1 = new SWFObject('player.swf','player','200','200','9'); 

	
s1.addParam('flashvars','file=video/video.flv&stretching=none');

	
播放器会在此元素里面。

	
	
sample2：当播放器窗口大于视频分辨率时，视频按照原尺寸播放，因此周围会显示出播放器的背景色

	
var s1 = new SWFObject('player.swf','player','600','600','9');
 
	
s1.addParam('flashvars','file=video/video.flv&stretching=none');

	
播放器会在此元素里面。

	
	
	

	
	
stretching=exactfit，填充满播放器，但不成比例播放

	
sample1：当播放器窗口小于视频分辨率时：

	
var s1 = new SWFObject('player.swf','player','200','100','9'); 

	
s1.addParam('flashvars','file=video/video.flv&stretching=exactfit');

	
播放器会在此元素里面。

	
	
sample2：当播放器窗口大于视频分辨率时：

	
var s1 = new SWFObject('player.swf','player','300','700','9'); 

	
s1.addParam('flashvars','file=video/video.flv&stretching=exactfit');

	
播放器会在此元素里面。

	
	
	

	
	
stretching=uniform，用黑边（即播放器背景）进行拉伸，保持原视频的宽高比

	
s1.addParam('flashvars','file=video/video.flv&stretching=uniform');

	
播放器会在此元素里面。

	
	
	

	
	
stretching=fill，类似uniform，但完全填充

	
s1.addParam('flashvars','file=video/video.flv&stretching=fill');

	
播放器会在此元素里面。
