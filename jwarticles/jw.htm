二、安装

1 上传文件：player.swf、swfobject.js、yt.swf，以及所需图片和视频

2 插入代码：（SWFOBJECT 1.5方法，其他方法不介绍，可参看000.htm）

<!―引入swfobject.js文件 -->

<script type='text/javascript' src='swfobject.js'></script>

 

<!-- 把播放器放到下面这个DIV元素里，用ID识别 -->

<div id='mediaspace'>This text will be replaced</div>

 

<!-- 全局设置JW player的各种参数 -->

<script type='text/javascript'>

     var s1 = new SWFObject('player.swf','player','400','300','9','#FFFF00');

     s1.addParam('allowfullscreen','true');

     s1.addParam('allowscriptaccess','always');

     s1.addParam('wmode','opaque');

     s1.addParam('flashvars','file=video.flv');

     s1.write('mediaspace');

 

三、配置

1 配置flashvars参数A：flashvars参数介绍及演示

var s1 = new SWFObject('player.swf','player','400','300','9','#FFFF00');

400和300代表swf容器的宽高值（像素），也可以用”100%”百分比表示

s1.addParam('allowfullscreen','true');

启用全屏模式，默认是false

s1.addParam('allowscriptaccess','always');

允许swf和外部脚本通讯（以及链接到外部页面），默认是sameDomain

s1.addParam('flashvars','file=video.flv');

指定swf文件所要加载的变量，可无限加载。更多flashvars参见adobe网站，常用的如：

l  bgcolor(#rrggbb) ：设置swf背景颜色

l  menu(true,false)：设置为false将隐藏大多数右键菜单

l  wmode(opaque,transparent,window)：设置swf的透明度，可用来适应z-index

s1.addParam(‘flashvars’,’file=playlist.xml&backcolor=white&autostart=true’);

用&分隔不同的flashvars，用=分隔名称和值，之间没有空格，且必须用小写。

s1.write('mediaspace');

同一页面中，通过ID和name识别

 

更多参数列表见：http://blog.deconcept.com/swfobject/

 

2 配置flashvars参数B：将flashvars配置信息保存在外部XML配置文件中（适用于SWFOBJECT 1.5）

s1.addParam(‘flashvars’,’config=config.xml’);

 

config.xml的结构是：

<config>

         <repeat>true</repeat>

         <backcolor>333333</backcolor>

         <volume>40</volume>

         <playlist>right</playlist>

         <playlist.size>150</playlist.size>

         <controlbar>over</controlbar>

</config>

 

注：如果在配置文件和嵌入的Flashvars里面同时有设置，那么会应用后者的设置。

 

3 配置播放列表（playlist）

播放列表属性包括：文件、图片、标题、叙述、作者、时长等等（一部片子该有的信息都有了）

完整的属性列表见：http://developer.longtailvideo.com/trac/wiki/Player5FlashVars#Playlistproperties

 

步骤1：选择其中一种播放列表格式

l  ASX feeds（例子：http://www.longtailvideo.com/jw/upload/asx.xml）

l  ATOM feeds使用Media扩展（例子：http://www.longtailvideo.com/jw/upload/atom.xml）

l  RSS feeds使用iTunes和Media扩展（irss例子：http://www.longtailvideo.com/jw/upload/irss.xml，mrss例子：http://www.longtailvideo.com/jw/upload/mrss.xml）

l  Basic SMIL feeds（例子：http://www.longtailvideo.com/jw/upload/smil.xml）

l  XSPF feeds（例子：http://www.longtailvideo.com/jw/upload/xspf.xml）

 

步骤2：创建播放列表（我们将使用XSPF feed作为演示），内容如下：

<playlist version=”1” xmlns=’http://xspf.org/ns/0/”>

<title>Example XSPF playlist for the JW Player</title>

<info>http://developer.longtailvideo.com/trac</info>

<trackllist>

     <track>

              <title>The First Video</title>

              <creator>LongTail Video</creator>

              <info>http://www.longtailvideo.com/</info>

              <annotation>The first video that will appear in the playlist.</annotation>

              <location>http://developer.longtailvideo.com/files/firstvideo.flv</location>

     </track>

     <track>

              <title>The Second Video</title>

              <creator>LongTail Video</creator>

              <info>http://www.longtailvideo.com/</info>

              <annotation>The Second video that will appear in the playlist.</annotation>

              <location>http://developer.longtailvideo.com/files/secondvideo.flv</location>

     </track>

</tracklist>

</playlist>

 

说明：

l  track标签：包围内容

l  title标签：内容标题

l  creater：内容创建人

l  info：内容原始地址

l  annotation：内容简介

l  location：内容实际位置

 

步骤3：插入JW Player命名空间

为了让所有的JW player文件属性能够应用到所有feed格式上，4.4版本的播放器引入了jwplayer 命名空间。只要在feed里插入它，那些feed格式自身不支持的文件属性（比如RSS不支持的provider或duration）就会被得到支持。还可以在播放列表中插入任何的FlashVar，用来自定义每个播放列表项目。然而，一些配置选项只会被加载一次。

 

注意最上行的标签，它使用xmlns属性定义了JW Player的命名空间，见下：

<playlist version=”1” xmlns=http://xspf.org/ns/0/ xmlns:jwplayer=http://developer.longtailvideo.com/trac/wiki/FlashFormats>

<title>…</title>

<info>…</info>

<tracklist>

     <track>

              <title>…</title>

              <creator>…</creator>

              …

              <jwplayer:author>…</jwplayer:author>

              <jwplayer:provider>http</jwplayer:provider>

              <jwplayer:duration>34</jwplayer:duration>

     </track>

     …

</tracklist>

</playlist>

 

步骤4 配置播放列表外观和行为

JW player的播放列表可高度定制：编辑播放列表的外观、如何附着到播放器、读取播放列表时播放器如何响应。以下给出几个FlashVars帮助你起步：

file (JW4) / playlistfile (JW5) (undefined):播放器启动时加载的XML播放列表的路径。

注：所有播放列表里的文件必须在同一个域名底下，这是Adobe的安全考虑。

 

比如：（SWFOBJECT 1.5）

s1.addParam(‘flashvars’,’file=playlist.xml&playlist=bottom&playlistsize=180’);

 

l  Playlist(bottom):播放列表的位置。值可以设置为：bottom，over，right或none。

l  playlistsize (180):设置为below指播放列表的高度，设置为right指宽度。

l  repeat (none):设置为list来一次播放所有视频；设置为always来不断播放song/video/playlist；设置为single来继续重复播放选中的视频。

l  item (0):播放列表项目应该开始播放。用它设置特定的起始播放项目。

5 配置插件

插件是用Actionscript 3写成的程序，通过Flash变量加载，插件是执行某种特殊功能的程序。插件是host在LongTail站点上，无需下载，引用即可。

方法一：用flashvars配置插件

pluginname.pluginvariable=variablevalue

方法二：用XML文件配置插件 ，配置文件的内容是：

<pluginname.pluginvariable>variablevalue</pluginname.pluginvariable>

详细的插件使用说明见：http://www.longtailvideo.com/support/addons/10629/using-jw-player-plugins

完整的插件参数列表见：http://www.longtailvideo.com/addons/

 

6 配置Media Provider

方法一：用flashvars配置Media Provider

mediaprovidername.mediaprovidervariable=variablevalue

方法二：用XML文件配置Media Provider，配置文件的内容是：

<mediaprovidername.mediaprovidervariable>variablevalue</mediaprovidername.mediaprovidervariable>

 

比如：

http.startparam ('start'):查询参数，提供给服务器起始点的位置

rtmp.loadbalance (false):如果设置为true，RTMP模块（JW4）或MediaProvider（JW5）会假定flashvar是XML文件（目前只支持SMIL），使用它来达到加载平衡目的。

fcsubscribe (JW4) / rtmp.subscribe (JW5) (false):如果设置为true，RTMP模块（JW4）或MediaProvider（JW5）会使用一个流式的订阅机制来加载平衡。

8 配置JW Box：JW Player Lightbox弹出浮动层

步骤1 下载

jquery.jwbox.js、jwbox.css、jquery.js

 

步骤2 插入代码

<script type=”text/javascript” src=”http://www.mydomain.com/jwbox/jquery.js”></script>

<script type=”text/javascript” src=”http://www.mydomain.com/jwbox/jquery.jwbox.js”></script>

<link rel=”stylesheet” type=”text/css” href=”http://www.mydomain.com/jwbox/jwbox.css” />

 

步骤3 配置JW box（SWFOBJECT）

<div class="jwbox">

         <img width="160" height="120" alt="JW Box Image" title="JW Box Image" src="[Thumbnail URL]"/>

         <div class="jwbox_hidden">

                   <div  class="jwbox_content">

                            <div id="myPlayer"></div>

                   <p>Click the close button to return to the web page.</p>

                   </div>

         </div>

</div>

<p>Click the image to display a player with a video in a JW Box.</p>

 

还可以在弹出层上插入图片和文字，见：http://www.longtailvideo.com/support/jw-player/21/jw-box-the-jw-player-lightbox

 

JW Box原理结构图：

<div class=”jwbox”>

         [Default display HTML]

         <div class=”jwbox_hidden”>

                   <div class=”jwbox_content”>

                            [Lightboxed HTML]

                   </div>

         </div>

</div>

 

说明：

<div class="jwbox"> 是JW Box的容器

[Default display HTML] 是要被点击从而弹出播放层的东西

<div class="jwbox_hidden"> 里面的东西直到点击[Default display HTML]才会显示

<div class=”jwbox_content”> 任何在jwbox_content外，jwbox_hidden内的东西都会被当作关闭按钮
